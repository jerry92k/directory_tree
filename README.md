# directory_tree
현재 디렉터리를 기준으로 디렉터리 트리를 구성하는 명령줄 프로그램
- f-lab 멘토링 숙제

## 프로그램 정의
- 파일은 출력하지 않고 디렉터리만 출력
- 디렉터리 방문은 DFS 알고리즘 사용
- Deque 인터페이스를 제공하는 컬렉션을 스택처럼 사용

## 예시
:(최상위에 A1등의 디렉터리들이 아래 깊이에 B1등이, 세번째 깊이에 C1등이 있다고 가정)

```
$ java tree
CWD: C:\TEMP
+-- A1
   +-- B1
      +-- C1
   +-─ B2
   +-- B3
+-- A2
```
